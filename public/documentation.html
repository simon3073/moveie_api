80s Epics

This web application allows access to information about epic movies from the 1980's. Users will be able to sign up, update their personal information, and create a list of their favorite 80's movies.


API Endpoints

GET

/movies

Get a list of movies

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data of all the movies

GET

/movies/info/[movie]

Get data about a movie

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data of the movie requested, for example: 


{
    "Title": "Film Title",
    "Year Released": "1987",
    "Director": [Director_ID],
    "Description": "Film Description"
    "imdbRating": 7.6,
    "imgURL": '...url.jpg',
    "Genre": [ 
        [GenreID], 
        [GenreID]
    ],
    "Actor": [
        [ActorID],
        [ActorID]
    ],
}

GET

/bio/[director]

Get information about Director

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data of all the movies, for example: 


{
    "Name": “John Hughes”,
    “Born” : "03-18-1950"
    "Died" : "08-6-2009",
    "Bio" : "A biography of the director",
    "moviesDirected": [
        [MovieID],
        [MovieID]
    ]
}

GET

/movies/genre/[genre]

Search for movies with in a Genre

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data about movies with a specific genre, including genre, for example: 


{
    “Genre”: “Action”,
    “Movies” : [
        "Commando",
        "Karate Kid",
        "Rambo"
    ]
}

GET

/movies/actor/[actor]

Search for movies starring an Actor

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data about movies with a specific actor, including actor, for example: 


{
    “Actor”: “Molly Ringwald”,
    “Movies” : [
        "The Breakfast Club",
        "Pretty in Pink",
        "16 Candles"
    ]
}

GET

/movies/rating/[rating]

Search movies with a certain IMDB Rating

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data about movies with an IMDB rating above what has been provided, including rating, for example: 


{
    “Rating”: “7”,
    “Movies” : [
        "Bull Durham"
        "Karate Kid",
        "Commando"
    ]
}

POST

/account

Create an account

Request Body Data Format

JSON Object holding data about a new user: 


{
    "Username" : "username",
    "Email" : "useremail@email.com",
    "Password" : "password2022",
    "Birthday" : "2000-01-01T00:00:00Z"
}

Response Body Data Format

JSON Object holding data about the new user added, for example: 


{
    "Username" : "username",
    "Email" : "useremail@email.com",
    "Password" : "password2022",
    "Birthday" : "2000-01-01T00:00:00Z"
    “Favourite-movies” : []
}

GET

/account/[username]

Get account details

Request Body Data Format

NONE

Response Body Data Format

JSON Object holding data about the logged in user, including an ObjectId, for example: 


{
    “id”: ObjectId("00000"),
    "Username" : "username",
    "Email" : "useremail@email.com",
    "Password" : "password2022",
    "Birthday" : "2000-01-01T00:00:00Z"
    “Favourite-movies” : [
        [MovieID],
        [MovieID]
    ]
}

PUT

/account/[username]

Update account details

Request Body Data Format

JSON Object holding data about a new user. Example: 


{
    "Username" : "testuserUpdate",
    "Email" : "testuser@email.com",
    "Password" : "justtesting99",
    "Birthday" : "1977-05-25T00:00:00Z"
}

Response Body Data Format

JSON Object holding data about the user updated, including an ObjectId, for example: 


{
    “id”: ObjectId("00000"),
    "Username" : "testuserUpdate",
    "Email" : "testuser@email.com",
    "Password" : "justtesting99",
    "Birthday" : "1977-05-25T00:00:00Z"
    “Favourite-movies” : [
        [MovieID],
        [MovieID]
    ]
}

DELETE

/account/[username]

Update account details

Request Body Data Format

NONE

Response Body Data Format

A text message indicating whether the account was successfully deleted

PUT

/account/movies/[movie]

Add a movie to a users favourites

Request Body Data Format

NONE

Response Body Data Format

A text message indicating whether the movie was successfully added from the favourites list

DELETE

/account/movies/[movie]

Remove a movie from a users favourites

Request Body Data Format

NONE

Response Body Data Format

A text message indicating whether the account details were successfully deleted/p>
