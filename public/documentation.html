80s Epics

This web application allows access to information about epic movies from the 1980's. Users will be able to sign up, update their personal information, and create a list of their favorite 80's movies.


API Endpoints

Request URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET NONE JSON Object holding data of all the movies
Get data about a movie /movies/info/[movie] GET NONE JSON Object holding data about a movie. Example: 

                        {
                            "Title": "Planes, Trains & Automobiles",
                            "Year Released": "1987",
                            "Director": ["John Hughes",
                            "Description": "A Chicago advertising man must struggle to travel home from 
                                            New York for Thanksgiving, with a lovable oaf of a shower 
                                            curtain ring salesman as his only companion."
                            "imdbRating": 7.6,
                            "imgURL": '...url.jpg',
                            "Genre": [
                                "Comedy", 
                                "Drama"
                            ],
                            "Actor": [
                                "John Candy",
                                "Steve Martin"
                            ]   
                        }
Get information about Director /bio/[director] GET NONE JSON Object holding bio data about director. Example 

                        {
                            “Director”: “John Hughes”,
                            “Born” : "03-18-1950"
                            "Died" : "08-6-2009",
                            "Movies directed": [
                                "Planes, Trains & Automobiles",
                                "16 Candles",
                                "The Breakfast Club"
                            ]
                        }
Search for movies with in a Genre /movies/genre/[genre] GET NONE JSON Object holding data about movies with a specific genre, including genre. Example 

                        {
                            “Genre”: “Action”,
                            “Movies” : [
                                "Commando",
                                "Karate Kid",
                                "Rambo"
                            ]
                        }
Search for movies starring an Actor /movies/actor/[actor] GET NONE JSON Object holding data about movies with a specific actor, including actor. Example 

                        {
                            “Actor”: “Molly Ringwald”,
                            “Movies” : [
                                "The Breakfast Club",
                                "Pretty in Pink",
                                "16 Candles"
                            ]
                        }
Search movies with a certain IMDB Rating /movies/rating/[rating] GET NONE JSON Object holding data about movies with an IMDB rating above what has been provided, including rating. Example 

                        {
                            “Rating”: “7”,
                            “Movies” : [
                                "Bull Durham"
                                "Karate Kid",
                                 "Commando"
                            ]
                        }
Create an Account /account POST JSON Object holding data about a new user. Example: 

                        {
                            "Name": "Test User",
                            "Email": "test@user.com",
                            "Password": "UserPassword-2020",
                        }
JSON Object holding data about the new user added, including an ID. Example 

                        {
                            “id”: 8,
                            “Name” : “John Smith”,
                            “Email” : “johnsmith@email.com”,
                            “Password” : “purpleMonkey-84”,
                            “Favourite-movies” : []
                        }
Get account details /account/[id] GET NONE JSON Object holding data about the logged in user, including an ID. Example 

                        {
                           “id”: 8,
                            “Name” : “John Smith”,
                            “Email” : “johnsmith@email.com”,
                            “Password” : “purpleMonkey-84”,
                            “Favourite-movies” : []
                        }
Update an Account /account/[id] PUT JSON Object holding data about a new user. Example: 

                        {
                            "Name": "Test User",
                            "Email": "test@user.com",
                            "Password": "UserPassword-2020",
                        }
A text message indicating whether the account details were successfully updated
Delete an Account /account/[id] DELETE NONE A text message indicating whether the account details were successfully deleted
Add movie to favourites /account/movies/[movie] PUT NONE A text message indicating whether the movie was successfully added from the favourites list
Remove movie from favourites /account/movies/[movie] DELETE NONE A text message indicating whether the account details were successfully deleted
